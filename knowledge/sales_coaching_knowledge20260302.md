# キャリドラ営業ナレッジベース：属性別商談攻略ガイド

> **目的**: 営業マンが商談前にお客様の年齢・職種・性別を入力するだけで、即座に気をつけるべきポイント・刺さる営業トーク・避けるべき地雷発言をAIがサジェストするためのRAGナレッジ
>
> **出典**: 中山魁星による営業フィードバック道場（属性別攻略講座5回＋個別FB指導23回）＋CS統括・長安大吉によるキャリドラ勉強会＋まさかアポ道場の全記録を体系化

## 第0章：キャリドラのサービス本質理解（全営業マン必読）

### 0-1. なぜサービス理解が重要か

車を売る営業マンは車について詳しい。「前に向いて走ります」しか知らない営業マンはいない。キャリドラを売る営業マンも、サービスの本質を深く理解していなければ、お客様への対応が変わらない。ただし、理解した内容を「全て伝える」必要はない。本質をわかった上で、お客様に合わせて対応することが重要。

### 0-2. キャリドラの顧客成功の定義

**キャリアドライバーシップを持ってもらうこと**＝キャリアの運転席を自分でハンドル握れる状態にすること。

- 転職すること自体はゴールではない
- 現職に残る選択をする人もいる
- 次の転職で経験を積んで、その次で理想に行く人もいる
- デザイナーになるために学校に通う進路を選ぶ人もいる
- 方向性は受講生ごとにバラバラ
- **一番大事なのは、自分の方向性を自分で決められる状態にしてあげること**

このゴールが理解できていれば、カウンセリングでの細かいトーク（ハウトゥー）に振り回されなくなる。

### 0-3. キャリドラと他社キャリアコーチングの違い

| 項目 | 一般的なキャリアコーチング | キャリドラ |
|------|------------------------|----------|
| 基本手法 | コーチング（答えを教えない、内省を促す） | コーチング＋ティーチング（ハウトゥーも丁寧にやる） |
| 仕事探し | 自分で考えてもらう | 業界・職種・企業の探し方を具体的にティーチング |
| 内定獲得 | サポート薄い | 書類対策・面接対策を実践的にトレーニング |
| エージェントとの違い | 求人紹介しない | 求人紹介しない。提携企業なし。全求人が対象 |
| 費用の仕組み | 受講者から費用をもらう | 受講者から費用をもらう（企業からは一切もらっていない） |

**営業トークへの示唆**: 「ライザップの転職版」という例えが刺さりやすい。「エージェントは企業からお金をもらっている→紹介先が限られる。キャリドラは受講者からお金をもらっている→提携企業なし→全求人が対象→100%あなたのために動ける」

### 0-4. セッションの全体構造

キャリドラの受講セッションは大きく4フェーズに分かれる。

**フェーズ1：自己分析（セッション1〜5）** → 軸を作る

| セッション | 内容 | 目的 |
|-----------|------|------|
| S1. 過去分析 | モチベーショングラフで価値観を発見 | 「何に価値を感じるか」の方向性を見つける |
| S2. 現状未来分析 | ギャップから理想と改善要素を洗い出す | ニーズの可視化。コンプレックス＝自分が欲しているものの裏返し |
| S3. 強み分析 | ストレングスファインダーで強みを発見 | 強み＝自然とやってしまうこと（≠得意なこと・できること） |
| S4. ライフプラン | 理想の人生設計とゴール設定 | 「次の転職で何を得るべきか」を逆算 |
| S5. 自分軸 | S1〜S4の総まとめで転職軸を作成 | 軸に基づいて仕事を探す準備完了 |

**フェーズ2：仕事探し** → 軸を使って探す
- 業界・職種研究：広いカテゴリーから絞り込む（恋愛で言えば「まずデートしてみる」）
- 企業研究：軸に照らして○×△で評価
- 「見た目（年収・休日）で選ぶ」のは「イケメンだから結婚する」のと同じ→軸で選ぶ

**フェーズ3：内定獲得** → 実践トレーニング
- 論理的思考力トレーニング（結論→根拠→まとめ）
- 書類対策（履歴書・職務経歴書）
- 面接対策（基礎→よく聞かれる質問→個別対策）

**フェーズ4：適職を育てる** → スタートダッシュ＋アドバンス
- **適職は「探す」以上に「育てる」方が大事**
- スタートダッシュアシスト：自分軸をベースに、入職後の出来事を内省→行動→ゴール設定のサイクル
- アドバンスプログラム：マネジメントスキルだけではない。営業スキル、時間管理等その人に必要なもの
- ブランドハプンスタンス理論：将来は100%予測できない→だからこそ自分軸が重要

### 0-5. コース別の違い（営業トークに必要な知識）

| コース | セッション回数 | 特徴 | 分割月額目安 |
|-------|-------------|------|-----------|
| ハイキャリア | 最少 | 内定戦略カリキュラムのみ。書類対策1回。面接対策3回 | 月2.1万円〜 |
| エキスパート | 中 | ＋論理的思考＋スタートダッシュアシスト。書類対策2回。面接対策5回 | 月2.9万円〜 |
| プロフェッショナル | 最多 | ＋アドバンスプログラム（マネジメントスキル等）。面接対策7回。サポート1年間 | 月4.4万円〜 |

**バリューアッププログラム**:
- A：ポータブルスキル獲得（コミュニケーション、ポジティブシンキング、タスクマネジメント、デザインシンキング、ライティング）
- B：補填用（過去分析やり直し、転職軸振り返り等、必要なセッションを追加）
- C：セルフアップ（自己否定型・カウンセリングが必要な受講生向け）

### 0-6. 「強み」の正しい理解（営業トークに直結）

受講生の多くは「強み」の意味を誤解している。営業マンも正しく理解しておくべき。

| よくある誤解 | 正しい理解 |
|------------|----------|
| 強み＝得意なこと・できること | 強み＝自然とやってしまうこと |
| 強み＝周りより優れている点 | 強み＝無意識にやっている行動パターン |
| 長所と短所は別もの | 長所＝強みが良い方向に働いている状態。短所＝強みが悪い方向に働いている状態 |

**営業トークへの示唆**: お客様に「強みがない」と言われた時、「強みって自然とやってしまうことなんです。やれてしまうから気づいていないだけで、それが活かせない環境にいるから辛いんです」と返すことで、キャリドラの価値を伝えられる。

### 0-7. 価値観の発見（過去分析の核心）

セッション1で使うモチベーショングラフでは、数値の違和感に「なぜ？」を深掘りすることで価値観が見える。

**例**: 「彼女に刺される」が-75なのに「睡眠障害」が-100 → なぜ？ → 刺される経験は滅多にない＝話のネタになる＝刺激的なことに価値を感じる → 価値観は「刺激・冒険」

**営業トークへの示唆**: お客様の発言の中で「なぜそう思うんですか？」を深掘りすることで、表面的な不満の裏にある本当のニーズ（価値観）を引き出せる。これはカウンセリングで行うことと同じスキル。

## 第0.5章：顧客流入経路と広告別インサイト

### 0.5-1. 広告と制約率の関係

キャリドラにはマーケチームが複数の広告を回しており、広告の種類によって顧客の期待値・制約率が異なる。

**基本原則**: 制約率が高い（決めやすい）顧客ほど広告費が高い。安くたくさん来る広告ほど制約率が低め。

利益（＝広告費あたりの制約率）が上がれば → 報酬原資が増える → 決めやすい広告に再投資できる → さらに良いアポが増える。制約率の向上は個人の年収にも直結する。

### 0.5-2. 「まさか」広告からの顧客（まさかアポ）

**広告の内容**:
「仕事辞めたいと思った時、まさか転職エージェント使おうとしてないよね？」→ エージェントはダメ → 無料で適職診断＋プロが相談 → ホワイト企業に転職できた → 年収も上がった → 試してみて

**まさかアポ顧客のインサイト（心理状態）**:
- 「エージェントかなぁ？違うのかなぁ？何だろう？」（サービスの正体が不明）
- 「無料で何かいいことしてくれるらしい」（無料期待）
- 「ホワイト企業に行けたらいいなぁ」（漠然とした理想）
- 「面倒なことはしたくない」（低関与）
- 「適職診断ってどんなんだろう」（診断への興味）

**重要**: Lステップでまさかアポかどうかを判別する方法は現状ない。したがって、全てのアポを「まさかアポかもしれない」前提で対応するスキルが必要。

### 0.5-3. まさかアポ攻略の核心

**失注パターン（よくある失敗）**:
- エージェントとの違いを「ロジカルに」説明するだけ → お客様にとって「それが嬉しいかどうか」が伝わっていない
- 「悩みを一緒に言語化していく」「今後の不安を一緒に考える」→ 正しいが、お客様にとってセクシーに聞こえない
- 「求人紹介はしないんです」→ 「じゃあ何しに来たんだ」と思われるリスク

**成功パターン**:
お客様にとっての「嬉しさ」「メリット」を先に伝えてからエージェントとの違いを説明する。

**成功トーク例（吉山スタイル）**:
「500名以上の方を見てきているんですけど、一回一回の転職ってすごく重要だから、適当なアドバイスはしたくないんですよね。だからしっかりと○○さんのことを知った上でお話ししたい」→ エージェントとの差別化が「お客様にとっての嬉しさ」として伝わる

**成功トーク例（図師スタイル - ライザップ比喩）**:
「イメージで言うとライザップの転職版です」→ 「あ、トレーニングするんだ」と一発で理解 → その上で「エージェントは企業からお金をもらっている。僕らは受講者からお金をもらっている。だから提携企業なし。全求人が対象。100%あなたのために動ける」

**核心**: 聞き手がどう思うかの**想像力**。自分がやっていることを正しく説明するのではなく、お客様にとってそれが「嬉しいかどうか」で伝える。

### 0.5-4. 汎用性の高さ

まさかアポ攻略スキルは全アポで使える。なぜなら、エージェントと勘違いしている（＝最も期待値が低い）お客様を救えるスキルがあれば、キャリドラを正しく理解しているお客様はもっと簡単に救える。

今までの属性別道場（20代前半/後半、営業職、事務職、女性）は「お客様の属性に合わせたアプローチ」。まさかアポ道場は「お客様の期待値に関わらず使える基盤スキル」。両方を組み合わせることで制約率が最大化する。

## 第1章：営業の基本フレームワーク（全属性共通）

### 1-1. 営業ファイブステップ

キャリドラのカウンセリング商談には5つのステップがある。どの属性の顧客であっても、このステップの順番を崩してはならない。ただし、各ステップでの「やり方」は属性によって大きく変わる。

| ステップ | 内容 | 目的 |
|---------|------|------|
| 1. 関係構築 | アイスブレイク・自己紹介・趣味の深掘り | 「好き」と「尊敬（または安心）」を勝ち取る |
| 2. 転職活動の必要性 | 今すぐ動く理由を気づかせる | 現状維持ではダメだと腹落ちさせる |
| 3. 一人での活動の限界 | 自力では難しい理由を示す | プロの支援が必要だと理解させる |
| 4. プロと一緒に人生を変える | キャリドラの価値を伝える | 「ここなら大丈夫」と思わせる |
| 5. サービス説明 | プラン・料金の提示 | 意思決定を促す |

### 1-1b. 営業マン自身の市場価値（最重要マインドセット）

メガネ屋さんに裸眼の店員はいない。ユニクロにザラを着た店員はいない。ライザップに太ったトレーナーはいない。ECCに英語が下手な先生はいない。

**キャリドラの営業マンが年収アップ・市場価値アップにコミットしていなかったら、このサービスは売れない。**

道場の目的は制約率を上げること → 制約率が上がると利益が上がる → 利益が上がるとみんなの報酬原資が増える → より良い広告に再投資できる → 決めやすいアポが増える → さらに利益が上がる。この好循環を回すことが、営業マン一人一人の年収アップに直結する。

### 1-2. 成長の法則（70:20:10）

アメリカ・シリコンバレーの調査によると、ビジネスパーソンの成長要因は「70%が現場での経験」「20%が上司からのフィードバック」「10%が研修」。道場で学んだことは今日のアポから即実践して検証すること。

### 1-3. 「場」を読む力

矛盾するフィードバックに混乱する営業マンが多いが、最適解は「場」によって変わる。場とは、自分の醸し出す雰囲気、声のトーン、お客様の性格、商談のどのタイミングか、それまでにどういう会話をしてきたか、これら全ての要素の組み合わせ。同じ質問に対する回答でも、場が違えば正解は異なる。

### 1-4. 実年齢 vs 精神年齢

全ての属性判断は「実年齢」ではなく「精神年齢・思考パターン」で行う。25歳でも20代後半型の思考の人がいれば、29歳でも20代前半型の思考の人もいる。年齢はあくまで初期仮説の入口に過ぎない。お客様の発言内容・反応パターンから、実際にどの属性カテゴリに近いかを随時修正する。

### 1-5. 制約率の分解公式（モテ理論）

「制約率を上げたい」は「モテたい」と同じぐらいバクッとしている。「よし、決めた。制約率を上げよう」と思っても制約率は上がらない。モテるために変数を特定するのと同じで、制約率も変数を分解して弱い部分を特定する必要がある。

**制約率の分解公式**:

制約率 ＝ ブリッジ率（初回→次回約束を取れる率）× 実施率（約束通り再カウンセリングが実施できる率）× クロ獲得率（クロージングの約束を取れる率）× クロ実施率（クロージングが実施できる率）× クロ制約率（クロージングで現地を取れる率）×（1 − クーリングオフ率）

モテの例えで言えば「年収もある程度あるし、年も若いし、身長も悪くない。でも決定的に清潔感がない」→ 清潔感を良くした方がいい。全部の数字が悪いわけじゃなくて、特定の変数だけが低い。そこを集中的に改善する。

### 1-6. 失注パターン診断

失注の仕方から課題を特定できる。

| 失注パターン | 考えられる課題 | 改善の方向 |
|-------------|--------------|-----------|
| 事前キャンセル・リスケが多い | 訴求がズレている | 嫌われてはいないが「大事な時間」と認識させられていない。訴求内容を見直す |
| 飛び（無断キャンセル）が多い | 関係構築に問題 | アイスブレイク・セットアップの質を見直す |
| ブリッジ（次回約束）が取れない | 関係構築＋訴求ズレの両方 | 一番伸びしろ満点。両方改善が必要 |
| クロージングで制約できない | 感情の機微のキャッチアップ不足 | トーク自体は問題ないケースが多い。お客様の不安・怖さなどの感情を汲み取るアドリブ力を鍛える |

### 1-7. 決着リードタイムの管理

初回アポから決着（制約or失注）までのリードタイム日数が重要。トップ営業マンは平均4日以内に決着。リードタイムが6日以内の営業マンはクロージング実施率が100%。長くなるほど実施率が下がり、事前キャンセル・飛びが増える。

**5分電話テクニック**（詳細は第5章参照）でリードタイムを短縮し、モチベーションを保つ。

### 1-8. 会話は「建築」である

塩崎式（ニアポスタイル）の核心。ただおしゃべりしているのではなく、序盤の会話が後ろのクロージングにつながっている。各ステップが独立しているのではなく、伏線回収のようにつながりがある。

**ただし最重要の注意点**: 「よし建築するぞ」と思ってるカウンセラーが目の前に来たら、お客様はどう感じるか？→ 「売られる」と感じる。特に女性アポでは逆効果。建築の意識はアポ外（録画分析・振り返り時）に持ち、アポ中はお客様に集中する。

### 1-9. 期日握り（アンカリングテクニック）

「いつまでに転職したいですか？」といきなり聞いても答えが出にくい。こちらから先に基準を示す（アンカリング）。

**トーク例**:
「今これだけしんどい状況ですよね。でも転職活動って、本当にぴったり合う会社をサクッと数週間で見つけられるかというと、そんなに甘くないんですよね。逆にそんなにサクッと内定取れる会社って怪しいと思った方がいいです。最低3ヶ月はかかると思った方がいい。でも長くやりすぎても良くなくて、今の○○さんの年齢だったら3ヶ月〜半年ぐらいが目安としてすごくいい。○○さんはじゃあ何ヶ月後ぐらいに、本当にぴったり合った職場につきたいですか？」

顧客の表情が「3ヶ月〜半年」で曇ったら「長くても1年とか」に調整。大事なのは、こちらから基準を見せてから聞くこと。

### 1-10. 第一印象は取り返せない

Zoom接続直後の第一印象は取り返しがきかない。「もう一回初めましてのノリで話させてください」はできない。

**チェックポイント**:
- 息混じりの声になっていないか（自信なさそうに見える）
- 微妙に前のめりになっていないか（圧迫感）
- 明るい表情・声のトーンで入れているか
- 自己紹介の声の張りは十分か

**スマホ横向き問題（重要な気づき）**: お客様にスマホを横向きにしてもらう場面で「ロック外して横に置くだけですよ」と言うのは、表面的には正しい指示だが、お客様が横向きにしたくない本当の理由は「部屋がいっぱい映る」「仕事終わりの疲れた顔を映したくない」等の可能性が高い。「資料を見せる時に横の方が見やすいですよ」「手で持ってると1時間疲れちゃうので、置いてもらった方がいいですよ」等、お客様の立場に立った提案をする。すごく小さいことだが、第一印象で「この人は私のことを考えてくれている」と思ってもらえるかどうかの分かれ目。

### 1-11. 事前情報（Lステップ）の活用

アポ前にLステップの情報を確認するだけで、商談の方向性を事前に立てられる。

**確認すべき情報**: 年齢、性別、既婚/独身、年収帯、転職回数、希望（できるだけ早く転職したい等）、流入経路（広告の種類）

**活用例**: 「35歳・独身・男性・年収400万・転職0回・できるだけ早く転職したい」→ 30代モード＋危機感訴求が有効。転職経験0回なので不安が強い可能性。年収400万は同年代比で低め→年収アップの訴求が刺さりやすい。

事前に方向性を立てておくと、アポ中のリラックス度が上がり、お客様への対応に余裕が生まれる。AI（道場くん）に事前情報を投げて方針案をもらうのも有効。

## 第2章：年齢別攻略法

### 2-1. 20代前半（22歳〜25歳目安）の顧客

#### 心理構造

20代前半の顧客は「将来の不安」が刺さらない年代である。地獄の未来（JM）や求人減少グラフなどの将来訴求は逆効果になる。22歳の顧客には「来年も求人はある」「まだ若いから大丈夫」と思える心理的余裕がある。したがって、この年代に響くのは「今この瞬間の痛み」である。

#### 制約率が低くなる3つの失敗パターン

**失敗①：未来の逃避で訴求してしまう**
将来の不安を煽っても「まだ若いから大丈夫でしょ」と受け流される。「10年後こうなりますよ」は響かない。

**失敗②：営業側の主観的な甘さ**
「この人まだ若いし大丈夫だろう」という営業マン側の思い込みで、深掘りが浅くなる。

**失敗③：ペライ承認**
「めっちゃいいですね」「頑張ってますね」だけの薄い承認では、自己肯定感が低い20代前半には逆効果。「そんなことないのに」と心の距離が離れる。

#### 効果的なアプローチ

**＜ヒアリング：今の痛みを徹底深掘り＞**

20代前半には「今この瞬間のしんどさ」を自分の口から語らせることが最重要。以下の質問を繰り返す。

- 「何が起きてるんですか？」
- 「その時どんな気持ちになるんですか？」
- 「どうして嫌だと思うんですか？」
- 「それによって何か影響出てますか？」
- 「それが2〜3年続いたらどうなりますか？」（今の痛みの増幅）

気持ちの部分を何度も聞くことで、お客様自身が「今のままではダメだ」と気づく。

**＜マインドブロック解消＞**

20代前半が動けない理由は以下の4つのいずれか。

| ブロック | 典型的な発言 | 解消の方向 |
|---------|------------|-----------|
| 自己理解不足 | 「自分に何が向いてるかわからない」 | 「だからこそプロと一緒に見つけよう」 |
| 怖さ | 「転職して失敗したらどうしよう」 | 「向いてるところが定まらないまま入社する方が怖くない？」 |
| お金 | 「お金がなくて動けない」 | 分割払い等の具体策を提示 |
| やり方がわからない | 「何から始めたらいいかわからない」 | 「それを一緒にやるのがキャリドラ」 |

質問例：「どうして今転職活動できてないんですか？」を丁寧に聞いて4つのうちどれかを特定し、「キャリドラだったら大丈夫だよ」と解消する。イメージとしては、扉の鎖を外してあげる感覚。しんどさから逃れたいのに動けない理由を消す。

**＜承認の仕方＞**

NG：「すごいですね」「頑張ってますね」（ペライ承認）
OK：「こういう状態でここをこうやってるのは、こういうことだと思います。そこが素晴らしい」

具体的事実で承認する。行動・選択・価値観を言語化して褒める。

**＜セルフクロージング＞**

20代前半は何度聞いても嫌がらない傾向がある。分厚くセルフクロージングして大丈夫。

- 「一番変えたい理由は何ですか？」
- 「どんな人が喜ぶと思いますか？」
- 「変わったら最初に何したいですか？」

**＜親ブロック対策＞**

20代前半は親ブロックが発生しやすい。初回アポ・再カウンセリング段階から扱うこと。

- 「親御さんに相談したりしますか？」を当たり前に聞く
- 親との関係性を把握し、伝え方・タイミングを指導する
- 「内定取ってから言った方がいい」「こういう言い方で伝えよう」と台本レベルまで教える
- 「親や友達に言われてやっても頑張れない。自分で決めることが大事」と強調する

#### 訴求の順番

① 今この瞬間のしんどさを明確化 → ② 今動けてない理由（マインドブロック）を解消 → ③ 親ブロック対策 → ④ セルフクロージング

#### 具体トーク例

顧客発言：「向いてるところが定まらないまま入社すると怖い」
→ 効果的な返し：「どういうところが怖いですか？」（怖さブロックの特定）

顧客発言：「集中力が続きにくい」
→ 効果的な返し：「それによって何か影響ありましたか？」（今の痛みの深掘り）

顧客発言：「自分を責めちゃう」
→ 効果的な返し：「それが2〜3年続いたらどうなりますか？」（痛みの増幅）

### 2-2. 20代後半（26歳〜29歳目安）の顧客

#### 心理構造

20代後半は「先延ばしが最も合理的に見える」年代。20代前半ほど挑戦のメリットは感じず、30代ほどの危機感もない。最大のインサイトは「うまくいきたい」ではなく「失敗したくない」である。

現状維持バイアスが強く、以下のような自己正当化の発言が出やすい。
- 「もう少し情報集めてから」
- 「まずは休むべき」
- 「どうせ失敗する」
- 「今じゃなくてもいいかな」

#### 効果的なアプローチ

**＜ステップ1：不安の正当化（最重要）＞**

動かない理由・言い訳を全部ウェルカムで出させる。絶対に否定しない。

- 「そう思うのは本気で考えてる証拠ですよ」と正当化する
- 「気持ちわかります」だけでなく、具体的に寄り添う
- 例：看護師の妻を持つ営業マンの場合「奥さんも同じ悩みを近くで見てるのでわかります」

**＜ステップ2：第三者トーク＞**

先延ばしにした人がどうなったかのストーリーを語る。

- 「同じように悩んで先延ばしにされた方がいて、その方は結局30代半ばで…」
- 斬新な手法：「その方は今の○○さんに多分こう言うと思います」と第三者に言わせる

**＜ステップ3：失敗リスクの低減＞**

「めっちゃうまくいく」ではなく「失敗しない」を見せる。

- 看護師の場合：「国家資格あるから戻れる。それがアドバンテージ」
- 「自己分析して適職わかるだけでも最悪じゃない」とハードルを下げる

**＜先延ばしが一番リスキーだと気づかせる訴求＞**

- 「先延ばしにしていったらこうなる。とんでもない失敗が待ち受けている」
- 「30代になって今の仕事しんどいからやめますってなった時、もう30歳で次が6社目。今より年収下がる」
- 「社数だけ増えてスキルは積み上がってなくて歳だけ重ねて、どんどん市場価値が下がっていく」
- 「35歳、40歳になってすごく後悔してこられてる方をいっぱい見てきている」
- そうなってほしくないからこそ→「でも正しいやり方知らなかっただけ」→「何百万種類とある会社の中で、20代の○○さんにどれも合ってませんっていう方が確率低くないですか？」

**＜ビッグチャンス×ラストチャンス＞**

- 「20代でよく来てくれた。これはビッグチャンスであるとともにラストチャンスでもある」
- 「今動き出すのが、実は一番合理的」

#### 訴求の違い（年代比較）

| 年代 | 訴求の核 |
|------|---------|
| 20代前半 | 「今この瞬間のしんどさ」から動かす |
| 20代後半 | 「先延ばしが実は一番リスキー」「失敗しない」を見せる |
| 30代 | 「今すぐ動かないと本当にやばい」危機感で動かす |

#### 地雷発言

- 「めっちゃうまくいきますよ！」→ 信用されない。「失敗しない」の方が響く
- 「今すぐやらないとヤバいですよ！」→ 20代後半には重すぎ。30代向けの訴求
- 過度な楽観論 → 「どうせうまくいかない」と防衛反応が出る

### 2-3. 30代の顧客

#### 心理構造

30代は「今すぐ動かないと本当にやばい」が響く年代。実際に市場価値の低下を実感し始めており、危機感訴求が有効。20代後半の「先延ばし型」とは異なり、「もう猶予がない」という切迫感で動く。

#### 効果的なアプローチ

- 危機感をしっかり見せる（JM＝地獄の未来が刺さる）
- 年齢による市場価値の低下をデータで示す
- 「この年齢だからこそ」「今が最後のチャンス」を強調
- 家族がいる場合は「家族のためにもならない」訴求が有効

#### 家族持ちへの訴求（30代頻出）

NG：「家族のために今の仕事続けてても意味ないですよね？」（いきなり否定）
OK：「ご家族のために自分を犠牲にして頑張ってこられたの、本当にすごいことだと思います。その責任感があるからこそだと思いますよね」（承認）→「あえてお聞きしたいんですが、合わないお仕事を2〜3年続けることって、お体を壊してしまったり、ご家族に辛い姿を見せることを考えると、結局ご家族のためにもならないんじゃないかなと思うんですが、○○さんはどう思われますか？」（攻撃ではなく質問として投げる）

## 第3章：性別別攻略法

### 3-1. 女性顧客の攻略法

#### 心理構造の本質（たった一つ）

**「私が大切にされているか」を常に敏感に察知している。**

女性顧客は営業マンの「括弧の中身」（本音・裏の意図）まで全部見透かす。男性顧客は括弧の中身を感じにくいが、女性顧客には全て見える。Zoom接続から切断まで、ずっと「大切にされてるか」をチェックしている。

#### 括弧の中身とは

営業マンの発言には必ず「表面の言葉」と「括弧の中身（本音）」がある。

- 表面：「飲み物ありますか？」
- 括弧（NG）：（台本に書いてあるから言っとこう）
- 括弧（OK）：（本当にリラックスしてほしい。初めてのカウンセリングだから安心してほしい）

女性顧客はこの括弧の中身を読み取る。

#### 失敗パターン

- 「本音出してもらうためにアイスブレイクしよう」→ 括弧が見えて逆効果
- 「今すぐ転職活動のイエスを取ろう」→ 「売られてる」と感じる
- ペライ承認：「めっちゃいいですね」は響かない

#### 効果的なアプローチ

**＜関係構築：「好き」と「安心」を勝ち取る＞**

通常の商談は「好き」と「尊敬」を勝ち取るが、女性顧客は「好き」と「安心」が重要。安心＝大切にされている感覚。

**＜ヒアリング：把握ではなく「共感と没入」＞**

- 「世界一の理解者」ではなく「協力者」「仲間」を目指す
- 事実より気持ちの部分を深掘り：「どんな気持ちになるんですか？」を何度も
- 感情に寄り添うことが最優先

**＜承認：具体的事実で過程を褒める＞**

NG：「すごいですね」「いいですね」
OK：「こういう状態でこうやってるのは、こういうことだと思います。そこが素晴らしい」

- 過程を褒める
- 価値観を承認する
- 「可愛い」はセンスや好みを褒めるニュアンスで使う（見た目ではなく）

**＜訴求：「べき論」は絶対禁止＞**

女性顧客に「べき論」は死ぬほど刺さらない。

NG：「今から動いた方がいいですよ」（べき論）
OK：「動いていいんだよ」（許可）

効果的な訴求フレーズ：
- 「今まで周りに配慮して我慢してきたよね。これからは自分を大切にしていいんだよ」
- ストーリーで語る（JMもTMもストーリーで共感させる）
- 自分で気づいて自分で決めた感覚を持たせる

**＜違和感への対応＞**

女性顧客は小さな違和感を見逃さない。以下のサインが出たら即座に処理する。

- トーンが変わる
- 生返事になる
- 間が空く

→ 「今ちょっと気になったことありますか？」とすぐ深掘りする
→ クーリングオフ・親ブロックを受けやすいので、積極的に懸念を聞く

**＜アイスブレイク＞**

証人（共通点）と仲間感が重要。

- 「私も同じところ行ったことある」「私もそれ好き」
- 相手の好きなものを一緒に好きになりにいく
- 趣味を深く聞く：「面白ポイントどこですか？」
- 「いい趣味してますね」と具体的に承認

**＜男性が苦手な女性顧客への対応＞**

男性カウンセラーに対して警戒心が強い女性顧客は一定数いる。この場合：

- 画面オンを強制しない（最初は「もしよかったら」のスタンス）
- 声のトーンを柔らかく（但し自信なさそうにならない）
- ヒアリングの深掘りを急がない（信頼が貯まるまで待つ）
- 「聞いちゃっても大丈夫そうですかね？」の確認を入れる
- フォーユートーク：「適当なアドバイスしたくない。○○さんのことをちゃんと知った上で、本当に合うお話をしたい」

**注意**: この属性はクロージングまでのハードルが高い分、制約後の満足度は高い傾向。焦らず丁寧に進めること。

**＜「理解者ポジション」の極意＞**

女性アポで最も重要なのは「理解者ポジション」を取ること。これは承認・共感・許可の三要素で構成される。

- 承認：具体的に、プロの視点で「あなたの○○は価値がある」
- 共感：「そう感じるのは当然」「わかる」
- 許可：「動いていいんだよ」「自分を大切にしていい」

「理解者」は「同意者」とは違う。お客様の言うことに全て「そうですね」と同意するのではなく、お客様の感情・価値観を深く理解した上で、プロとして正しい方向を示す存在。

## 第4章：職種別攻略法

### 4-1. 営業職の顧客

#### 心理構造

**「売られたと思われた瞬間、試合終了」**

営業職の顧客は自分も営業だから、営業されることに極めて敏感。感情に寄り添われると「だるい」と感じる。ロジカル・ファシリテーション型が響く。

#### 制約率の特徴

営業職顧客の制約率は他職種と比べて異常に低い（工場勤務の半分以下、月によっては1/3）。営業職道場実施後は4%上昇。

#### 3つの原則

| 領域 | 通常のアポ | 営業職アポ |
|------|----------|-----------|
| ヒアリング | 感情を深掘り | **環境構造**を深掘り |
| リアクション | 共感・理解 | **ファシリテーション（仕切り）** |
| 訴求 | 危機感 | **機会損失** |

**＜ヒアリング：感情よりも環境構造＞**

NG：「どんな気持ちになるんですか？」（感情深掘り）
OK：「どういう時にしんどいんですか？」「どういうことがあった時にそう思うんですか？」

事実・環境・構造を握りにいく。「電話する時？お客様と話す時？上司と話す時？どういう時に嫌だと思いますか？」と場面を具体化する。

**＜リアクション：共感よりもファシリテーション＞**

NG：「めちゃくちゃわかります」「どんな気持ちですか？」の繰り返し
OK：「なるほど、そうなんですね。確かにこういう状態だったら僕でもしんどくなりますね。じゃあこういうことで悩んでるってことですね」

一回受け止めて、要約してまとめる。

ファシリテーション＝仕切り。「今からこういう話をしていくよ」「あなたが知りたいことはこれだよね。じゃあそれに沿って説明するね」というスタンス。

**＜ファシリの具体例：ビジネスモデルの質問への対応＞**

営業職顧客から「御社ってどうやって利益得てるんですか？」と聞かれた場合：

NG：いきなり「キャリアの専門学校です。エージェントはこうで…」と説明する
OK：
1. まず受け止める：「そこ気になりますよね。皆さんにもよく聞かれるところなので、今お伝えしますね」
2. 仕切りを入れる：「確かにそこはどうなってるんだろうと思いますよね」
3. 順を追って説明する
4. **最後に相手にとってのメリットで締める**：「だから私たちは100%○○さんの思いや価値観、○○さんらしい働き方を実現していきたいと思ってこれをやってるんです」

ポイント：この質問の本質は「御社の利益が心配」ではなく「私がお金を払うのか？どうやってメリットを得られるのか？」。表面の質問の奥にある本当の関心事に答える。

**＜訴求：危機感よりも機会損失＞**

NG：「今すぐやらないとヤバいですよ！」（危機感訴求）
OK：「この環境構造のままだと、○○さんが本来持っている力を発揮できない機会がどんどん失われていく」（機会損失訴求）

比較的ロジカルに、データ・事実・構造で納得させる。

#### 営業職 × 女性の場合

女性で営業職の場合は、両方の要素を掛け合わせる。「大切にしてあげながらファシリする」。安心感を醸し出しつつ、仕切り型で進行する。非営業職の女性のように「共感だけ」では不十分だが、営業職男性のように「仕切りだけ」でもダメ。

### 4-2. 事務職の顧客

#### 心理構造

**「自分には武器がない」という強烈な思い込み**

事務職顧客は「市場価値がない」「転職してもうまくいかない」という自己否定ブロックが強い。現職の不満も言語化されにくく、理想も出てこない（年収も控えめに言う傾向）。

#### 制約率の特徴

数は多いが制約率が低い。得意な営業マンが少ない（芝、和田祐二、山本桃子が頭一つ抜けている）。

#### 3つのマインドブロック

| ブロック | 実態 | 対応 |
|---------|------|------|
| 自分には武器がない | 「ルーティンしかできない」「強みがない」 | ルーティンを才能として承認する |
| 年収アップへの感度 | 安定志向かと思いきや、選ばせると年収アップを選ぶ | 潜在ニーズとして意識する |
| 地雷が多い | 正しい順番で進めないと即終了 | ステップを丁寧に踏む |

#### 効果的なアプローチ

**＜ルーティンを才能として承認する＞**

NG：「頑張ってますね」
OK：「ミスなく業務を進める正確性は、何百人と見てきた中でも○○さんの強みとしてアピールできます」

プロの立場から具体的に言語化する：
- 「継続力が相対的に高い」
- 「納期を守る責任感が高い」
- 「他の職種でも見せ方次第で通用するスキル」

**＜潜在ニーズの引き出し＞**

事務職顧客の傾向を知った上で仮説を持ってヒアリングする。意外と年収アップ・キャリアアップに響く。選択肢を提示して選ばせると本音が出やすい。

**＜地雷を避ける正しい順番＞**

① マインドブロック理解 → ② 潜在ニーズ把握 → ③ 地雷回避 → ④ 訴求

自己否定ブロックが強いので、まず承認から入ること。いきなり訴求は絶対にNG。

#### 地雷発言

- 「今のスキルだと厳しいですね」→ 自己否定を強化して即終了
- 「事務職って市場価値低いんですよ」→ 追い打ち
- いきなりの危機感訴求 → ブロックが強化されるだけ

### 4-3. その他の職種への応用

#### 工場勤務

- 比較的素直に話を聞く傾向がある
- 制約率は営業職より高い
- 「御社ってどうやって利益得てるんですか？」のようなリテラシー系の質問は少ない
- 標準的なアプローチ（感情深掘り・共感・寄り添い）が効きやすい

#### 看護師

- 国家資格があるため「失敗しても戻れる」が安心材料になる
- 不規則勤務のストレスが深い痛みになっていることが多い
- 「戻れるアドバンテージがある」ことを失敗リスク低減の根拠にする

#### 教師

- 知的レベルが高く、ロジカルな説明が好まれる傾向
- 「ブルーロック」等の漫画のストーリーを使った訴求も有効な場合がある
- 元教師は話が合う（共通言語が使いやすい）

#### 施工管理

- 非営業職として女性の場合「共感型」が有効
- 「御社ってどうやって利益得てるんですか？」とは聞かないが「エージェントだと思って来た」はある
- その場合は共感→きっかけヒアリング→キャリドラの価値提示の流れ

## 第5章：商談シーン別の対応マニュアル

### 5-1. 「エージェントだと思って来た」への対応（まさかアポ攻略を含む）

**大前提**: 「まさか」広告（詳細は第0.5章参照）から来るお客様は「エージェントかなぁ？何だろう？」レベルの認識。この対応を全アポの基本にすることで、制約率が底上げされる。

**＜NG: ロジックだけで違いを説明する＞**

「悩みを一緒に言語化していく」「今後の不安を一緒に考えていく」→ 正しいが、お客様にとってそれが「嬉しいこと」として伝わっていない。「うちは求人紹介じゃないんです」→ 「じゃあ何しに来たんだ」リスク。

**＜OK: お客様にとっての「嬉しさ」を先に伝える＞**

**パターンA：ライザップ比喩（最もシンプル）**
「イメージで言うとライザップの転職版です」→ 一発で「トレーニングするサービス」と理解 → その上でエージェントとの構造の違いを説明

**パターンB：500名の実績＋フォーユー（吉山スタイル）**
「500名以上見てきているんですけど、一回一回の転職ってすごく重要だから、適当なアドバイスはしたくない。だからしっかり○○さんのことを知った上でお話ししたい」

**パターンC：営業職顧客向け（ファシリモード）**
1. 受け止め：「そこ気になりますよね。皆さんにもよく聞かれるところです」
2. 仕切り：「今からそこを説明させていただきますね」
3. 説明：キャリアの専門学校→エージェントとの違い→紹介料の構造上の問題
4. **メリットで締める**：「だから100%○○さんにとってベストオブベストのキャリアを一緒に作っていく。それが私たちです」
5. 安心感：「今日は営業じゃなくカウンセリング。売り込むつもりは一切ない」

**パターンD：非営業職・女性顧客向け（共感モード）**
1. 共感：「そうですよね〜。エージェントだと思って来てくださる方、めちゃくちゃ多いんですよ」
2. きっかけ深掘り：「そうやって転職考えたきっかけって何だったんですか？」（ここで共感・寄り添い）
3. 柔らかい説明：「私たちは求人紹介というよりも、○○さんにとって一番ベストな働き方って何かを一緒に考えていくっていう方がイメージ近いかもしれません」
4. エージェントの構造説明（柔らかく）
5. 安心感：「このタイミングで来てくれてよかったです。本当に」

**核心**: 全パターンに共通するのは「お客様にとって嬉しいこと」を先に伝えてから構造の説明に入ること。ロジックだけでは心は動かない。

### 5-2. 料金・ビジネスモデルを聞かれた時

**基本方針**: 質問の表面ではなく、「この人は何を気になってこの質問をしているのか」を考える。「御社の利益が心配」ではなく「私が払うの？どうやってメリット得るの？」が本質。

**順序**:
1. 受け止め＋仕切り（「気になりますよね。説明しますね」）
2. キャリドラの立ち位置の説明
3. エージェントとの構造的な違いの説明
4. **お客様にとってのメリットで必ず締める**（ここを忘れると印象がマイナスで終わる）
5. 「営業じゃなくカウンセリングです」「売り込む気は一切ない」で安心感

### 5-3. クロージングで「営業か？カウンセリングか？」と聞かれた時

これは諸説ある問題。正解は「誰が言うか」と「場」で変わる。

- 関係が深まり友達レベルの信頼がある場合 → 「確かに営業です」と腹を割ってもOK
- 初対面〜浅い関係で営業職の警戒心が高い場合 → 「カウンセラーとしてフラットに」が安全
- 重要なのは括弧の中身。「売りたい」が透けたら何を言っても無駄

**プッシュ型の罠（重要）**: 断りにくい空気を出してしまう営業マンは要注意。「断れない性格のお客様」から無理に制約を取ると、クーリングオフや事後キャンセルが増える。売れる時はバチコーンと売れるが、身内（お客様の家族や友人）の不信を買いやすい。プッシュ型ではなく、お客様自身が「やりたい」と思える状態を作ることが本質。

### 5-4. 再カウンセリング（ブリッジ）時の対応

初回で制約に至らなかった顧客との再カウンセリングでは：

- 前回のカウンセリングで何を話したかを復習してから臨む
- 「次回も聞きたいと思った理由」の原稿を必ず取っておく（これが取れていないと再カウンセリングの質が大幅に下がる）
- 日が空くとモチベーションが下がるため、5分の電話で繋ぎ止める提案が有効
- 再カウンセリングのブリッジが薄い場合、初回の訴求が不十分だった可能性が高い

**検討扱いのルール**: 明確に興味付けがまだ浅くて翌月になる場合は「検討」ではなく「失注」で報告する。「シフトが出てないので次回日程が決まらない」程度なら検討扱いでOKだが、1ヶ月先の再連絡は実質今月の契約に繋がらない。

### 5-5. 日程切れ・先延ばしリスクへの対応

**5分電話テクニック**:
シフト待ちなどで次回日程が決まらない顧客には「がっつり話す時間はシフト出てから取りましょう。ただ、5分だけ電話でお話しする時間をもらえませんか？」と提案。

フォーユーの理由づけ：
- 「忙しいとキャリアのことって後回しにしがち。モチベーション保つためにも」
- 「人気で予約いっぱいなので、電話で調整した方が最短で進められます」
- 「○○さんの大事な話をもっと聞きたい。そのための時間をちゃんと確保したい」

**運用の注意**:
- アポとアポの間の30分の空き時間で5分電話を入れる（負担にならない）
- 柄系（高単価狙い）の日に20時からの5分電話を入れるとクロージングに入れなくなるので避ける
- Zoomで画面オフで話すのもOK

### 5-6. 現地回収の定義とルール

「現地回収」＝クロージングから3日以内にコース決定・支払い方法確定・サイン完了すること。

- クロージングで「やります」→次のZoom（3日超）で手続き → これは「検討」扱い
- 3日以内に完了 → 「現地回収」

現地回収が遅れるとクーリングオフリスクが上がるため、極力早く完了させる。

### 5-7. クロージング時の料金提示

**料金提示の順番**（上位コースから提示する）:
1. プロフェッショナルコース（最上位）の内容と金額を先に見せる
2. エキスパートコース
3. ハイキャリアコース（最もリーズナブル）
4. 分割払いの月額を見せる
5. 「きついです」という反応が出たら、サポート期間が短いシンプルコースを提示

**重要**: 自ら単価を下げにいくムーブは避ける。お客様が「必要だ」と感じているなら、上位コースを勧めることはフォーユーでもある。ただし、制約率を安定させるために下位コースを積極提案する戦略も否定はしない。

### 5-8. アウト返し（反論処理）の重要性

クロージング時にお客様から出る「アウト」（反論・懸念・断り文句）への返し方が制約率を左右する。

**アウト返しが弱いと起きること**:
- クーリングオフリスクが上がる（納得しきれないまま制約してしまう）
- 事後キャンセルが増える
- 親ブロック・パートナーブロックで覆される

**よくあるアウトと返し方の原則**:
| お客様のアウト | 返しの方向性 |
|--------------|------------|
| 「お金がない」 | 分割の提案＋「投資」フレーム。生涯年収での回収見込みを示す |
| 「考えさせてください」 | 何を考えたいのか具体化させる。「何が引っかかってますか？」 |
| 「家族に相談しないと」 | 承認→「ご家族にどう伝えますか？」一緒に伝え方を考える |
| 「自分で転職活動できそう」 | 「できると思います。ただ、正しいやり方を知らずにやると…」リスク訴求 |

### 5-9. 訴求のズレを防ぐ

訴求がズレるとお客様が「ざらつく」。ざらついた瞬間から信頼が崩れ始める。

**典型的な訴求ズレの例**:

- 「軸がないですよね」と言ったが、お客様は自分なりの軸を持っている人だった → ザラつき発生
- 「今すぐ動かないとヤバいですよ」と言ったが、お客様はすでに行動する意志がある → 上から目線に感じる
- 「年収上げたいですよね」と決めつけたが、お客様の優先順位は「働き方」だった → 的外れ

**防ぎ方**: ヒアリングで十分に聞いてから訴求する。思い込みで決めつけない。お客様の言葉を使って返す。

### 5-10. 「適当なアドバイスしたくない」トーク

初回のセットアップで非常に有効なトーク。お客様との信頼関係を一気に構築できる。

**トーク例**:
「正直、まだ○○さんと初めましてで、ゲームが好きとか、ダイビングの資格を持ってるぐらいしか僕は分かってない。その状態で適当なアドバイスをしたくないんですよ。○○さんだったらこうした方がいいよって本当に言えるようになるために、いろいろ聞かせてもらってもいいですか？」

→ 「めっちゃ聞いてくるやん」と思われる前に許可を取れる
→ 「この人はちゃんと自分のことを知ろうとしてくれている」と感じてもらえる
→ ヒアリングの深掘りに対する抵抗感がなくなる

## 第6章：コミュニケーション技術

### 6-1. 承認の技術（全属性共通）

#### ペライ承認（絶対NG）

- 「すごいですね」
- 「頑張ってますね」
- 「めっちゃいいですね」

→ 自己肯定感が低い顧客ほど「そんなことないのに」と距離が離れる

#### 具体的承認（推奨）＝「プロとして具体で承認する」

**フォーマット**: 「こういう状態で、ここをこうやっているのは、こういうことだと思います。そこが素晴らしい」

これは「めっちゃ刺さる」レベルの承認であり、苦手な営業マンが多い。しかし、マスターすれば全属性で制約率に直結する最重要スキルの一つ。

**承認が苦手な人の自己診断**（タイプによって改善方法が異なる）:

| タイプ | 症状 | 改善方法 |
|-------|------|---------|
| 思いつかない型 | お客様の話を聞いても承認ポイントが浮かばない | ヒアリング力を鍛える。「どういう状態で、何をしているか」を聞く訓練 |
| ブレーキ型 | 承認ポイントは思いつくが「言ったら意味ないんじゃ」と躊躇する | 自分でブレーキをかけている自覚を持つ。まず言ってみる。お客様の反応を見る |
| 恥ずかしい型 | 具体的に褒めるのが照れくさい | 「プロとしての見解」というフレームで言う。個人の感想ではなく専門家の評価として伝える |
| テンプレ型 | 「すごいですね」しか出てこない | 具体的な事実＋価値観の言語化のフォーマットを使う |

**コツ**: ヒアリングしながら同時進行で承認する。お客様の日々の生活に思いを馳せて、「どういう状態のなかでそれをやっているのか」を言語化する。

例：
- 事実の指摘：「ご家族のために自分を犠牲にして、合わないと分かっていながらも続けてこられた」
- 価値観の承認：「それは○○さんの責任感だったり、そういうところがあるからこそだと思いますよね」
- 能力の言語化：「ミスなく業務を進められる正確性」「継続力が相対的に高い」
- 行動の言語化：「慣れ親しんだ場所を何度も離れて、その地域の風習に合わせてやってこれたって、それ相当な適応力ですよね」
- 環境への共感付き承認：「この仕事内容で（具体的な状況）をやっているって、それだけでもすごいことだと思います」

### 6-2. 質問の技術

#### 今の痛みを掘る質問（20代前半向け）
- 「何が起きてるんですか？」
- 「その時どんな気持ちになるんですか？」
- 「どうして嫌だと思うんですか？」
- 「それによって何か影響出てますか？」
- 「それが2〜3年続いたらどうなりますか？」

#### マインドブロックを探る質問
- 「どうして今転職活動できてないんですか？」
- 「何があったら動き出せそうですか？」
- 「一番引っかかっているのはどこですか？」

#### 環境構造を掘る質問（営業職向け）
- 「どういう時にしんどいんですか？」
- 「どういうことがあった時にそう思うんですか？」
- 「電話する時？お客様と話す時？上司と話す時？どういう時に嫌だと思いますか？」

#### 気持ちを掘る質問（女性向け）
- 「どんな気持ちになるんですか？」
- 「その時ってどういう感じでした？」
- 「リフレッシュできてますか？」

### 6-3. リアクションの技術

#### 共感型リアクション（女性・20代前半・非営業職向け）
- 気持ちに寄り添い、感情を受け止める
- 「いやぁ、思いますよね」「そうだよね」
- 大切にされている感覚を与える

#### ファシリテーション型リアクション（営業職向け）
- 一回受け止めて要約してまとめる
- 「なるほど、そうなんですね。確かにこういう状態だったら僕でもしんどくなりますね。じゃあこういうことで悩んでるってことですね」
- 相手が知りたいことを整理して順番に説明する（仕切り）

### 6-4. 第三者トークの技術

ストーリーで語ることは全属性で有効だが、特に以下で効果的。

- **20代後半**: 先延ばしにした人の末路ストーリー→「その方は今の○○さんに多分こう言うと思います」
- **女性**: JMもTMもストーリーで共感させる。べき論ではなくストーリー
- **家族持ち30代**: 「同じように悩まれた方がいて…」で第三者に語らせる

### 6-5. 恋愛の例え話テクニック

仕事と恋愛は本質がほぼ同じ。カテゴリーが違うだけ。仕事の例え話が思い浮かばない時は恋愛で例えるとお客様に伝わりやすい（CS統括・長安大吉の知見）。

| 仕事の概念 | 恋愛の例え |
|-----------|----------|
| 年収だけで転職先を選ぶ | 「イケメンだから結婚する」のと同じ。中身見ないの危なくない？ |
| 軸を持って転職活動する | 「自分が結婚相手に求めるもの」を明確にしてから出会いを探す |
| 業界・職種をまず広く見る | まずデートしてみる。デートして初めてその人のことがわかる |
| 自己分析で価値観を知る | 「自分がどういう人と一緒にいると幸せか」を知ること |
| 適職を育てる | 結婚してからが本番。日々の関係を育てていく |
| 強みが活かせない環境 | 自然体でいられない相手と付き合ってる状態→辛いに決まってる |

## 第7章：ニアポスタイル（塩崎式）の攻略法

### 7-1. ニアポ vs 三アポの違い

| 項目 | 三アポ（従来） | ニアポ（塩崎式） |
|------|--------------|----------------|
| 初回→制約までの回数 | 3回（初回→再カウンセリング→クロージング） | 2回（初回→クロージング） |
| 訴求方法 | 第三者ストーリーや自己ストーリーベースで語る | ストーリーは使わず、お客様自身の発言から合意を積み上げる |
| ファイブステップ | 同じ | 同じ（本質は変わらない） |
| 合意形成 | 各ステップで合意を取る | 18項目の合意チェックシートで体系的に取る |
| 決着リードタイム | 長め | 短い（トップは平均4日以内） |

**重要**: ニアポでも三アポでも、ファイブステップの本質は同じ。「関係構築→転職活動の必要性→一人での限界→プロと一緒に→サービス説明」の順番は変わらない。

### 7-2. 18項目の合意形成

ニアポスタイルでは、商談フローチェックシートの18項目を一つずつお客様と合意していく。各項目が独立しているのではなく、前のステップが後ろにつながる「建築」構造になっている。

**よくあるミス**:
- 合意項目の言い方がズレている（「軸がない」という訴求をしたが、お客様は軸がある人だった→ザラつきが発生）
- 期限を握れていない（いつまでに転職活動したいかの合意がない）
- 合意の確認を飛ばして先に進んでしまう

### 7-3. ニアポの難しさ

ニアポは三アポよりセンスが問われる部分が大きい。特に以下が難しい：

- 第三者ストーリーを使わずに「今すぐ動くべき」と判断してもらう
- 「プロを使った方が合理的」と納得してもらうロジック構築
- 初回で十分な関係構築＋訴求＋ブリッジを完了させる時間管理

## 第9章：AIサジェストのための属性マッチングルール

### 9-1. 入力情報と初期マッチング

AIツールが顧客の属性情報を受け取った場合、以下の優先度で攻略法をマッチングする。

**優先度1：職種**
営業職 → 営業職モード（ファシリ型）を発動
事務職 → 事務職モード（承認先行型）を発動
その他 → 標準モード

**優先度2：性別**
女性 → 女性モード（安心・大切にされてる感）を掛け合わせる
男性 → 標準

**優先度3：年齢**
22-25歳 → 20代前半モード
26-29歳 → 20代後半モード
30歳以上 → 30代モード

**掛け合わせの例**:
- 27歳・女性・事務職 → 20代後半 × 女性 × 事務職 → 「不安を正当化＋安心感＋ルーティンを才能として承認＋失敗しない訴求＋べき論禁止」
- 24歳・男性・営業職 → 20代前半 × 営業職 → 「今の痛み深掘り＋環境構造ヒアリング＋ファシリ型リアクション＋マインドブロック解消」
- 32歳・女性・看護師 → 30代 × 女性 → 「危機感訴求＋安心感＋国家資格のアドバンテージ＋大切にされてる感」

### 9-2. 商談中の発言によるモード切替シグナル

| 顧客の発言 | シグナル | 推奨アクション |
|-----------|---------|--------------|
| 「御社ってどうやって利益得てるんですか？」 | リテラシー高い。ファシリモードへ | 仕切り型の説明→メリットで締め |
| 「エージェントだと思って来ました」 | 期待値のズレ | 共感→きっかけ深掘り→キャリドラの価値提示 |
| 「まだ若いし大丈夫かなと…」 | 20代前半思考 | 今の痛み深掘りに切替 |
| 「失敗したらどうしよう」 | 20代後半思考（失敗回避型） | 不安正当化→失敗リスク低減 |
| 「もう少し考えてから…」 | 先延ばしバイアス | 第三者トーク→先延ばしリスク訴求 |
| 「自分に強みがない」 | 事務職型自己否定 | ルーティンを才能として承認 |
| 「親に相談しないと…」 | 親ブロック | 親との関係性ヒアリング→伝え方指導 |
| 「家族がいるので…」 | 家族を理由とした現状維持 | 承認→「家族のためにもならない」を質問形式で |
| トーン変化・生返事 | 違和感シグナル（特に女性） | 即座に「気になったことありますか？」 |
| 「面白いと思う仕事をしたい」 | 理想はあるが漠然としている | 環境構造ヒアリングで具体化→「それを見つけるのがキャリドラ」 |
| 「シフトがまだ出てないので…」 | 日程先延ばしリスク | 5分電話テクニックで繋ぎ止め |
| 「いくらぐらいかかりますか？」（早い段階で） | 料金への警戒心 | 先に価値を伝えてから料金提示。「学費」というフレームで説明 |
| 「今の仕事、商材はいいんですけど…」 | 言語化が甘い不満 | 「何がいいと思ってるか」を具体化させて、本当にそうか問う |
| 大きな夢・理想を語る（マクラーレン購入など） | 理想と現実のギャップが大きい | 否定せず受け止め→「現状の延長上にそれはありますか？」 |
| 「エージェントさんもう使ってます」「他でも相談してます」 | 比較検討中。まさかアポの可能性大 | エージェントとの違いをメリット先行で伝える（第0.5章参照） |
| 「で、何してくれるんですか？」「どういうサービス？」 | サービス理解がゼロ。まさかアポ確定 | ライザップ比喩→メリット先行→構造説明の順 |
| 「適職診断って何ですか？」 | 広告の適職診断に興味で来た | 「診断の先にあるもの」を伝える。診断はきっかけ、本質はキャリアドライバーシップ |
| 「無料って聞いたんですけど」 | 無料期待で来ている | 「今日のカウンセリングは無料」と安心させた上で、価値を伝えてから有料サービスの説明 |

### 9-3. 営業マンの課題診断ルール

AIが営業マンの数値データから課題を特定する場合の診断ロジック：

| 指標 | 閾値 | 診断 | 推奨アクション |
|------|------|------|--------------|
| ブリッジ率（初回→次回約束率）が低い | 全体平均以下 | 関係構築 or 訴求ズレ | 初回アポのアイスブレイク〜訴求を重点的に見直し |
| 実施率（約束→実施率）が低い | 全体平均以下 | 訴求の弱さ。「大事な時間」認識不足 | 訴求内容の見直し＋5分電話での繋ぎ止め |
| クロ獲得率（クロージング約束率）が低い | 全体平均以下 | 再カウンセリングの質が低い | 再カウンセリングでの訴求・セルフクロージング強化 |
| クロ実施率が低い | 全体平均以下 | 事前キャンセル多い→訴求ズレ / 飛び多い→関係構築問題 | 失注パターンで切り分けて対処 |
| クロ制約率が低い | 全体平均以下 | 感情キャッチアップ不足 | お客様の不安・怖さの扱い方を改善 |
| 決着リードタイムが長い | 6日超 | 日程切れリスク高 | 期日握り＋5分電話での短縮 |
| 事前キャンセル率が高い | - | 訴求がズレている | 「嫌われてはいないが重要性が伝わっていない」 |
| 飛び率が高い | - | 関係構築に問題 | アイスブレイク・セットアップ・第一印象の改善 |
| クーリングオフ率が高い | - | 違和感の処理不足 or 親ブロック対策不足 | 商談中の懸念処理＋親ブロック対策の早期実施 |

### 9-4. 絶対に避けるべき地雷発言（全属性共通）

| 地雷カテゴリ | 具体例 | なぜダメか |
|------------|--------|----------|
| ペライ承認 | 「すごいですね」「頑張ってますね」 | 自己肯定感低い人に逆効果 |
| いきなりのべき論 | 「今すぐ動くべきです」 | 特に女性・20代後半に刺さらない |
| 過度な楽観 | 「絶対うまくいきますよ！」 | 20代後半の「失敗したくない」に矛盾 |
| 相手の判断を否定 | 「その仕事を選んだのが間違いです」 | 全属性でNG。攻撃と受け取られる |
| 年齢を武器にした脅し | 「もう○歳ですよ？」 | 20代前半には響かない。30代には攻撃 |
| 括弧が透ける発言 | 台本通り感が強い「飲み物ありますか？」 | 特に女性が括弧を読み取る |
| スキル否定 | 「今のスキルだと厳しいですね」 | 事務職の自己否定を強化 |

## 付録：属性別クイックリファレンスカード

### 20代前半

```
【核心】今この瞬間の痛み → マインドブロック解消 → 親ブロック対策
【質問】「何が起きてる？」「どんな気持ち？」「それが続いたらどうなる？」
【承認】具体的事実で（ペライ承認禁止）
【訴求】未来の不安× → 今の痛みの深掘り○
【注意】セルフクロージングは分厚くOK / 親ブロック初回から対処
```

### 20代後半

```
【核心】先延ばしが一番リスキーだと気づかせる
【質問】不安を全部出させる → 「そう思うのは本気の証拠」と正当化
【承認】不安を正当化することが承認になる
【訴求】「失敗しない」を見せる / 第三者トーク / ビッグチャンス×ラストチャンス
【注意】「めっちゃうまくいく」は逆効果 / 楽観論は防衛反応を呼ぶ
```

### 30代

```
【核心】危機感が響く年代。今動かないと本当にやばい
【質問】将来の具体的リスクを一緒に見る
【承認】家族のための努力を認める
【訴求】JM（地獄の未来）が刺さる / 市場価値低下データ
【注意】家族持ちには「攻撃ではなく質問」で「家族のためにもならない」を気づかせる
```

### 女性

```
【核心】「大切にされているか」を常に見ている
【質問】気持ちの深掘り重視 / 「どんな気持ちになるんですか？」
【承認】過程・価値観を具体的に / ペライ承認は距離が離れる
【訴求】べき論禁止 → 「動いていいんだよ」（許可型）/ ストーリーで語る
【注意】括弧の中身まで見透かされる / 違和感は即処理 / 安心が最優先
```

### 営業職

```
【核心】売られたと思われた瞬間死ぬ
【質問】環境構造を掘る（感情より事実・場面）
【承認】ファシリ型（要約→まとめ→方向性提示）
【訴求】機会損失型 / ロジカル / データ・事実・構造
【注意】感情寄り添いは「だるい」と思われる / 仕切ることが喜ばれる
```

### 事務職

```
【核心】「自分には武器がない」という思い込みの解除
【質問】傾向を知った上で仮説を持つ / 年収アップに実は響く
【承認】ルーティン＝才能 / 正確性・継続力・責任感をプロの目線で言語化
【訴求】承認が十分にできてから（地雷多い）/ いきなり訴求は絶対NG
【注意】「スキルが低い」系の発言は自己否定を強化して即終了
```

### 難しい顧客パターン

```
【大きな夢を語る顧客（例：4500万の車が欲しい）】
→ 否定しない。一緒にワクワクする。ただし「現状の延長上にそれありますか？」で現実を見せる
→ テンプレ承認ではなく「相手に合わせた承認」が必須（頭フル回転させる）

【申込理由が多い顧客】
→ ポジティブシグナル。複数の動機がある人は制約率が高い傾向
→ 「一番刺さった理由」を深掘りしてクロージングの伏線にする

【既にキャリドラをよく知っている顧客】
→ 「よく調べてますね」で承認→「逆にどういうところに興味を？」で動機を深掘り
→ 説明を省略しすぎず、プロならではの視点を加える

【地方在住・転居不可の顧客】
→ リモートワーク可能な職種提案が刺さる
→ 「好きな時間・好きな場所で働ける」フレームが有効
```

## 付録B：道場の実績データ（攻略法の有効性を示すエビデンス）

属性別攻略道場の実施前後で、営業部全体の制約率に明確な変化が確認されている。

| 道場名 | 実施時期 | 効果 |
|-------|---------|------|
| 20代前半道場 | 1月中旬 | 20代前半の制約率が底上げされた |
| 20代後半道場 | 1月下旬 | 全年代で最下位だった20代後半が2月に一気にトップクラスに |
| 営業職道場 | 12月中旬 | 半年間低迷していた営業職の制約率が4ポイント上昇 |
| 女性道場 | 1月上旬 | 女性顧客への対応力が向上（個人差あり） |
| 事務職道場 | 2月上旬 | 事務職の制約率改善を目指して実施 |

**中山魁星のコメント**: 「攻略方法が分かっちゃうとすぐ数字出してくれる。だからどんどん道場をやって、みんなの時給単価を上げていきたい」

## 付録C：営業マンセルフチェックリスト

商談前に確認すべき項目：

**1. 顧客属性の確認**
- 年齢（20代前半/後半/30代）
- 性別
- 職種
- 既婚/独身
- 転職回数

**2. 自分の直近データ確認**
- ブリッジ率は全体平均以上か？
- 実施率は全体平均以上か？
- クロ制約率は全体平均以上か？
- 決着リードタイムは6日以内か？
- 事前キャンセルが増えていないか？

**3. 環境チェック**
- Zoom映り（照明はオレンジ寄り？服は明るい色？）
- 第一印象の声のトーン
- 前のめりになりすぎていないか？
- 息混じりの声になっていないか？

**4. マインドセット**
- 「建築しに行くぞ」ではなく「この人のことを知りたい」
- 「売りたい」ではなく「この人に合う選択肢を提示したい」
- 括弧の中身がフォーユーか確認
