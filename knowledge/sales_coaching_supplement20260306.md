# テーマA：キャリドラのサービス本質理解

## A-1. なぜ営業マンにサービス理解が必要か

車を売る営業マンは「前に向いて走ります」しか知らないということはない。エンジン、燃費、安全性能まで詳しい。キャリドラを売る営業マンも同じ。サービスの本質を深く理解してこそ、お客様に合わせた柔軟な対応ができる。

ただし重要なのは「理解した内容を全て伝えること」ではない。本質をわかった上で、お客様の状態に合わせて必要な部分だけ伝えること。これはキャリドラのトレーナーがセッションをやる時と全く同じ考え方。

## A-2. キャリドラの顧客成功の定義

**キャリアドライバーシップを持ってもらうこと。**

キャリアの運転席を自分でハンドル握れる状態にすること。これがキャリドラの顧客成功の唯一の定義。

転職することはゴールではない。受講生の方向性は一人一人バラバラ。

- 現職に残ることを選ぶ人もいる
- 次の転職で経験を積んで、その次の転職で理想に行く人もいる
- デザイナーになるためにデザインの学校に通う進路を選ぶ人もいる
- 転職活動中にキャリアについて考えた結果、結婚を優先すると決めた人もいる

一番大事なのは「自分の方向性を自分で決められる状態にしてあげること」。このゴールが腹落ちしていれば、カウンセリング中の細かいトーク（ハウトゥー）に振り回されなくなる。

## A-3. サービスの全体構造（4フェーズ・セッション詳細）

### フェーズ1：自己分析（セッション1〜5）＝「軸を作る」

| セッション | 名称 | 一言で言うと | 具体的にやること |
|-----------|------|-----------|-------------|
| S1 | 過去分析 | 価値観を見つける | モチベーショングラフで人生の出来事をスコアリング → 数値の違和感に「なぜ？」を深掘り → 価値観（何に価値を感じるかの方向性）が見える |
| S2 | 現状未来分析 | ギャップからニーズを洗い出す | As-Is / To-Beの整理。ギャップ・ネガティブ・コンプレックスにこそニーズが隠れている。価値観と紐づけると納得感が深まる |
| S3 | 強み分析 | 自分の強みに気づく | ストレングスファインダー実施 → 「強みとは何か」の認識を正しく修正 → エピソードと紐づけて実感させる |
| S4 | ライフプラン | 理想の人生を設計する | ゴール設定。次の転職で何を得るべきかを逆算。理想を考えられない受講生が多いので一緒に考える |
| S5 | 自分軸 | S1〜S4の総まとめ | 価値観・ギャップ・強み・ライフプランから転職軸を言語化。この軸で転職活動を進める |

### フェーズ2：仕事探し＝「軸を使って探す」

- 業界・職種研究：広いカテゴリー（業界・職種）から探す → 軸に照らして○×△で評価
- 企業研究：軸に当てはまりそうな業界・職種の中から、具体的な企業を調べる
- 探す力を身につけさせる：キャリドラが企業を紹介するのではなく、自分で探せる状態にする（キャリアドライバーシップ）

### フェーズ3：内定獲得＝「実践トレーニング」

- 論理的思考力トレーニング：結論→根拠→まとめの順で話せる・書ける状態にする
- 書類対策：履歴書・職務経歴書の作成（ハイキャリア1回、エキスパート以上2回）
- 面接対策：基礎→よく聞かれる質問→個別対策（受講生の転職活動状況に合わせて臨機応変に）

**他社との最大の違いはここ**。一般的なキャリアコーチングはコーチング（答えを教えない・内省を促す）のみ。キャリドラはコーチングに加えてティーチング（具体的なハウトゥー）も丁寧にやる。

### フェーズ4：適職を育てる＝「スタートダッシュアシスト＋アドバンス」

**「適職は探す以上に育てる方が大事」**（最重要コンセプト）

- スタートダッシュアシスト（3回）：入職後に自分軸をベースに内省→行動→ゴール設定のサイクルを回す
- アドバンスプログラム：その人に必要なスキルを学ぶ。マネジメントスキルはアドバンスの一部に過ぎない。営業スキル、時間管理、仕事管理、目的達成、組織把握、自分商品化など

**ブランドハプンスタンス理論**：将来のことを100%わかって人生設計できる人はいない。「わからないこと」も計画に入れておく。だからこそ自分軸が重要。軸があれば、予想外の出来事が起きても「何を基準に行動するか」が決まる。

## A-4. コース別の詳細

| コース | 内定戦略 | 論理的思考 | スタートダッシュ | アドバンス | バリューアップ | 面接対策 | サポート期間 |
|-------|---------|----------|-------------|---------|------------|---------|-----------|
| ハイキャリア | ○ | - | - | - | - | 3回 | 短い |
| エキスパート | ○ | ○ | ○ | - | - | 5回 | 中 |
| プロフェッショナル | ○ | ○ | ○ | ○ | - | 7回 | 1年間 |

**バリューアッププログラム**（別途付加）:
- A：ポータブルスキル獲得（コミュニケーション、ポジティブシンキング、タスクマネジメント、デザインシンキング、ライティング）
- B：補填用（過去分析のやり直し、転職軸の振り返りなど、必要なセッションを追加で4回分）
- C：セルフアップ（自己否定が強い・カウンセリング的対応が必要な受講生向け）

**営業トークへの示唆**：
- 面接が苦手な人 → エキスパート以上を推奨（面接対策5回以上）
- 書類をがっつりやりたい人 → エキスパート以上 or ハイキャリア＋バリューアップB
- プロフェッショナルとバリューアップBの組み合わせ → 補填セッションが計10回になるので結構多い。その場合はバリューアップAにして補填はアドバンスで対応する提案もあり

## A-5. 「強み」の正しい理解

受講生の多くは「強み」を誤解している。営業マンが正しく理解しておくべき最重要概念。

| よくある誤解 | 正しい理解 |
|------------|----------|
| 強み＝得意なこと・できること | 強み＝自然とやってしまうこと |
| 強み＝周りより優れている点 | 強み＝無意識にやっている行動パターン |
| 長所と短所は別のもの | 長所＝強みが良い方向に働いている状態。短所＝強みが悪い方向に働いている状態。同じ強みの表と裏 |

**具体例**：共感性が強い人 → マネージャーとしてメンバーの気持ちを察して声をかけられる（良い方向＝長所）。一方で、保険営業では「お客様に嫌がられるかも」と気にしすぎて辛い（悪い方向＝短所）。

**営業トークへの示唆**：
- 「強みがない」と言う事務職の顧客に → 「強みって自然とやってしまうことなんです。やれてしまうから気づかないだけ。それが活かせない環境にいるから辛いんです」
- カウンセラーが「それ強みですよ」と言っても、受講生は「ええ、これが強みなの？」と思いながら「そうなんですね」と言うだけ。同じ言葉でも前提の理解が違うと伝わらない。だから「強みとは何か」をちゃんと認識修正してあげることが重要。

## A-6. 価値観の発見方法（過去分析の核心）

セッション1で使うモチベーショングラフ：人生の出来事に-100〜+100のスコアをつけてもらう。

**発見のコツ**：数値に「違和感」がある箇所に「なぜ？」を深掘りすると価値観が見える。

**実例1**：「彼女に刺される」が-75、「睡眠障害」が-100
→ 「なぜ刺されるのが睡眠障害より軽いの？」 → 「滅多にない経験。話のネタになった」 → この人は「刺激・未知の体験」に価値を感じる → だからアメリカ旅行が+100（全てが未知の体験）

**実例2**：京都大学主席合格が+30、ボート大会出場が+100
→ 「なぜ京大主席なのに30？」 → 「仲間と一緒にやるのが好き」 → 価値観は「博愛・仲間」

**実例3**：大学受験失敗で母親が倒れた → -100
→ 「私ができなかったせいで」 → 達成・権力の価値観（周りの人よりできている自分でいたい）

**営業トークへの示唆**：
カウンセリング中のヒアリングでも同じスキルが使える。お客様の発言に「なぜそう思うんですか？」を深掘りすることで、表面の不満の裏にある本当のニーズ（価値観）を引き出せる。

## A-7. 恋愛と仕事の類似性（例え話テクニック）

仕事と恋愛は本質がほぼ同じ。カテゴリーが違うだけ。仕事の例え話が浮かばない時は恋愛で例えるとお客様に一発で伝わる。

| 仕事の概念 | 恋愛の例え |
|-----------|----------|
| 年収だけで転職先を選ぶ | イケメンだから結婚する。中身見ないの？後悔しない？ |
| 軸を持って転職活動する | 自分が結婚相手に求めるものを明確にしてから出会いを探す |
| 業界・職種をまず広く見る | まずデートしてみる。デートして初めてその人のことがわかる |
| 自己分析で価値観を知る | 自分がどういう人と一緒にいると幸せかを知ること |
| 適職を育てる | 結婚してからが本番。日々の関係を育てていく |
| 強みが活かせない環境 | 自然体でいられない相手と付き合ってる状態 → 辛いに決まってる |
| 企業研究で中身を調べる | 気になる人の情報収集。どのポイントを見ればいいか |
| 面接対策 | デートで好印象を与える準備。ただし嘘はバレる |

## A-8. セッションは「型」であり「正解」ではない

セッション1ではこれをやる、セッション2ではこれをやる、と型は存在する。しかし：

- 「セッション1だから過去分析をやらなきゃ」が先行すると受講生を置いてきぼりにする
- 「面接対策1ではこれをする」と決めつけると、転職活動フェーズの受講生のニーズに合わない
- スタートダッシュアシスト・アドバンスは特に臨機応変。結婚を決めた人のニーズをセッションシートで解決するのは不可能

**大事なのは一つだけ**：受講生にキャリアドライバーシップを持ってもらうこと。セッションはそのための道具。道具に使われてはいけない。

# テーマB：広告別の顧客インサイトと「まさかアポ」攻略

## B-1. 広告と制約率と利益の循環モデル

キャリドラのマーケチームは複数種類の広告を回している。広告の種類によって、来るお客様の期待値・制約率が異なる。

**基本構造**：
- 制約率が高い（決めやすい）お客様ほど → 広告費が高い
- 安くたくさん来る広告ほど → 制約率がやや低め

**利益循環モデル**：
制約率が上がる → 利益が上がる → 報酬原資が増える（営業マンの年収UP）→ 決めやすいアポを呼ぶ広告に再投資できる → さらに良いアポが増える → さらに利益が上がる → 強い営業マンの採用にも投資できる → チーム全体のレベルUP

**中山魁星のコメント**：「利益が全部を癒す。個人も会社も。これは資本主義・市場原理の話。」

## B-2. 営業マン自身の市場価値コミットメント

メガネ屋さんに裸眼の店員はいない。ユニクロにザラを着た店員はいない。ライザップに太ったトレーナーはいない。ECCに英語が下手な先生はいない。トヨタのディーラーにマツダで出勤する営業マンはいない。カルティエの店員がカルティエをつけていないのはヤバすぎる。

**障害年収をぶち上げるサービスを売っている営業マンが、自分の年収アップ・市場価値アップにコミットしていなかったら、このサービスは売れない。**

道場の目的は「営業マンにとっての年収アップのパーソナルトレーナー」として、制約率を通じてみんなの年収を上げること。

## B-3. 「まさか」広告とは何か

「まさか」は動画広告のタイトル。Instagram・TikTok・YouTubeショートで配信されている。

**広告の内容（要約）**：
「仕事辞めたいと思った時、まさか転職エージェント使おうとしてないよね？」→ エージェントはダメ → 適職診断の後に無料でキャリアのプロが相談に乗ってくれた → ホワイト企業に転職できた → 年収も上がった → 面倒なことしたくない人は試してみて

**特徴**：キャッチーで安価。たくさんのお客様が来てくれる。しかし制約率がやや低い。

**Lステップで判別不可**：現状、このお客様が「まさか」広告から来たかどうかをLステップで判別する方法はない。

## B-4. まさかアポ顧客の心理状態（5つのインサイト）

まさか広告を見てカウンセリングに来るお客様は、以下のような心理状態にある。

| # | インサイト | 詳細 |
|---|----------|------|
| 1 | サービスの正体が不明 | 「エージェントかなぁ？違うのかなぁ？何だろう？」レベル |
| 2 | 無料期待 | 「無料で何かいいことしてくれるらしい」と思っている |
| 3 | 漠然とした理想 | 「ホワイト企業に行けたらいいなぁ」程度の解像度 |
| 4 | 低関与 | 「面倒なことはしたくない」（広告がそう訴求している） |
| 5 | 診断への興味 | 「適職診断ってどんなんだろう」（入口は診断への好奇心） |

## B-5. まさかアポの失注パターン（よくある失敗）

**失敗1：ロジックだけでエージェントとの違いを説明する**
「僕らは悩みを一緒に言語化していく」「今後の不安を一緒に考える」→ 正しいが、お客様にとってそれが「嬉しいこと」として伝わっていない。自分たちがやっていることの正確な説明 ≠ お客様が聞いて嬉しい話。

**失敗2：「求人紹介しないんです」を先に言う**
→ 「じゃあ俺何しに来たんだ」と思われるリスク。お客様はエージェント（＝求人紹介してくれる場所）だと思って来ている可能性が高い。

**失敗3：年収アップの数字を先に出す**
「年収平均87.7万円上がってます」「月7〜8万円上がります」→ 数字自体はインパクトがあるが、信頼関係がない段階で出しても「営業トーク」としか受け取られない。

**失敗4：2リットルのペットボトルを見せる**
→ 物理的な見た目の問題。プロフェッショナルな印象を損なう小さなことが積み重なる。

## B-6. まさかアポの成功パターン

**核心**：聞き手がどう思うかの「想像力」。自分がやっていることを正しく説明するのではなく、お客様にとってそれが「嬉しいかどうか」で伝える。

**成功パターン1：ライザップ比喩（図師スタイル）**

「イメージで言うとライザップの転職版です」→ お客様が一発で「トレーニングするサービスなんだ」と理解 → その上で「エージェントは企業からお金をもらっている。僕らは受講者からお金をもらっている。だから提携企業なし。全求人が対象。100%あなたのために動ける」

**成功パターン2：500名実績＋フォーユー（吉山スタイル）**

「500名以上の方を見てきているんですけど、一回一回の転職ってすごく重要だから、適当なアドバイスはしたくないんですよね。だからしっかり○○さんのことを知った上でお話ししたい」→ エージェントとの差別化が「お客様にとっての嬉しさ」として伝わる。本質的にはやっていること同じだが、伝え方で印象が全然違う。

**成功パターン3：セットアップでの安心感**

「今日は営業でもなんでもないので、リラックスしながらお話しできたらと思ってます」「サービスを決めてもらうとか提案するとかいう時間ではないのでご安心ください」→ 構えを解く

**重要な注意**：「うち求人紹介しないですよ」は、言い方次第。「言われた方が嬉しい言い方」で伝える。「エージェントさんは企業からお金もらってるから紹介先が限られる。僕らはそうじゃないから、日本で求人が出る会社は全部対象です」→ 「紹介しない」がデメリットではなくメリットに変わる。

## B-7. まさかアポ攻略の汎用性

まさかアポ攻略スキルは、全てのアポで使える。

エージェントと勘違いしている（＝最も期待値が低い）お客様を救えるスキルがあれば、キャリドラを正しく理解しているお客様はもっと簡単に救える。

今までの属性別道場（20代前半/後半、営業職、事務職、女性）は「お客様の属性に合わせたアプローチ」。まさかアポ道場は「お客様の期待値に関わらず使える基盤スキル」。基盤スキル × 属性別アプローチ の掛け算で制約率が最大化する。

# テーマC：直近FB道場の実践的コーチング知見（2026年2月27日〜3月4日）

## C-1. スマホ横向き問題（第一印象での想像力）

**状況**：Zoom商談の冒頭でお客様にスマホを横向きにしてもらおうとした場面。

**NGな対応**：「ロック外して横に置くだけですよ」と手順を教える。

**なぜNG**：お客様が横向きにしたくない本当の理由は「やり方がわからない」ではなく、「横向きにすると部屋がいっぱい映る」「仕事終わりの疲れた顔を大きく映したくない」等の可能性が高い。

**OKな対応**：「資料をお見せする時に横の方が見やすいですよ」「手で持ってると1時間疲れちゃうので、置いてもらった方がいいかもしれません」→ お客様の立場に立った理由づけ。

**教訓**：すごく小さいことだが、第一印象は取り返しがきかない。初手で「この人は私のことを考えてくれている」と感じてもらえるかどうかの分かれ目。

## C-2. ネガティブ感情の裏に強みが隠れている

お客様がネガティブな感情を語る時、その裏には強みが隠れていることが多い。

**パターン**：
- 「給料低いんです」→ 年収への不満は単純に受け止め。「そこ変えたいですよね」
- 「仕事でやる気ない患者さんばかりでしんどい」→ 裏にある強み：「患者さんにちゃんと向き合いたい」という高い志
- 「マニュアル整ってなくて辛い」→ 裏にある強み：「仕組みを整えて効率よくやりたい」という組織化力
- 「パワハラがしんどい」→ 環境の問題。受け止めるだけ。

**活用法**：「しんどい」の中身を分類して、環境の問題はそのまま受け止め、仕事の中身へのしんどさは「裏に強みがある」可能性を見る。強みとして具体承認してあげると、自己肯定感が上がり、訴求が刺さりやすくなる。

## C-3. スシロー比喩（承認が訴求の前提条件である理由）

自分に強みがない・武器がないと思っているお客様に対して、「じゃあ環境変えましょう」「こういうキャリアがありますよ」と言っても響かない。

**スシロー比喩**：お腹いっぱいの人にマグロを出しても、サーモンを出しても反応は同じ。かっぱ巻きが嫌でお腹いっぱいなんじゃなくて、そもそも食欲がない状態。

**解決策**：まず食欲を出させる＝「あなたにはこういういいところがあるんだよ。こういう強みがあって、環境さえ変われば輝く可能性があるんだよ」と具体的に承認する。

**順番が超重要**：「あなたの良さをわかってくれていない人」から「あなたは自分の人生の主人公ですよ」と言われても「まあそうだけど…」で終わる。具体承認で「この人は私のことをわかってくれている」と感じた後に言うから響く。

## C-4. ポジティブループ概念（訴求の核心ロジック）

お客様が今の仕事で「ポジティブループ」が回っているかどうかが、訴求の出発点。

**ポジティブループとは**：強みを発揮 → お客様・同僚・上司に喜ばれる → 成長を実感 → 年収も上がる → さらにモチベーションが上がる → さらに強みを発揮…というサイクル。

**最強の訴求ロジック**：

「ポジティブループが回っている人は、わざわざ転職活動の無料カウンセリングに来ない。来ている時点でループが回っていないことはほぼ確定。回っていないなら、僕らカウンセラーは絶対に価値提供ができる。だって、ポジティブループが回った方がいい人生になるのは間違いないから。」

**7年勤続者への応用**：「7年も続けてきたのに、今キャリアのことで悩んでいる。ポジティブループが回っていたら、そうはならない。だから今が動くベストタイミング。」

## C-5. 着地を決めないコミュニケーション

プライドが高い・素直じゃない・人の話を聞かないタイプのお客様に対して。

**NGマインド**：「ここでJM（地獄の未来）を作ってやろう」「ここで自己分析大事って言わせてやろう」「今のまま2〜3年続けたら体壊しますって言わせてやろう」→ 着地を決めたコミュニケーションは、お客様に見透かされる。

## C-6. 承認スキルの4タイプ別診断と改善法

承認が苦手な営業マンは多い。しかし「苦手」の中身は人によって違い、改善法も異なる。

| タイプ | 症状 | 改善法 |
|-------|------|-------|
| 思いつかない型 | お客様の話を聞いても承認ポイントが浮かばない | ヒアリング力を鍛える。「どういう状態の中で、何をしているか」を聞く訓練 |
| ブレーキ型 | 思いつくが「言っても意味ないんじゃ」と躊躇 | 自分でブレーキをかけている自覚を持つ。まず言ってみてお客様の反応を見る |
| 恥ずかしい型 | 具体的に褒めるのが照れくさい | 「プロとしての見解」というフレームで伝える。個人の感想ではなく専門家の評価 |
| テンプレ型 | 「すごいですね」しか出てこない | 具体的な事実＋価値観の言語化フォーマットを使う |

## C-7. Being（あり方）vs Behavior（言動）の概念

**問題**：同じフィードバックを何度もらっても改善しない営業マンがいる。

**皇居ランニング比喩**：「綺麗なフォームで走ろう」と思って皇居を100周しても改善しないなら、200周すれば改善するのか？→ 多くの場合そうではない。フォーム（Behavior＝言動）だけを変えようとしても、走る目的や感覚（Being＝あり方）が変わっていなければ改善しない。

**Behavior（言動）だけ変える例**：「アイスブレイクで共感と承認を多めにしよう」→ アイスブレイクしか改善されない

**Being（あり方）を変える例**：「感情にフォーカスした質問をどんどんしていく自分になろう」→ アイスブレイクもヒアリングもクロージングの不安解消も全部変わる

**教訓**：5回10回練習してもできないなら、練習不足を疑うのではなく「そもそもの捉え方・あり方を間違えている」可能性を疑う。採用面接を通っている時点でポテンシャルは認められている。できないのは能力不足ではなく、フレームのズレ。

## C-8. 30代以降の訴求補足知見

**即戦力問題**：30代以降は採用市場で即戦力しか求められにくい。今と全然違う分野に行くなら年齢は気にした方がいい。だから早く動いた方がいい。

**キャリアの原理原則**：スキルと経験は積み上がっていくもの。早く入れば積み上げ期間が長くなる。3年遅れたら、後ろの年収千万円分の3年分のロスになる。

**「なぜ今なのか」の訴求**：「来年でもなく再来年でもなく今動いた方がいい。どんな人でも。ポジティブループが回っていないなら、今すぐ動くのが合理的。」

## C-9. Lステップ事前情報の活用

アポの前にLステップの情報（年齢、性別、年収帯、転職回数、希望、申込理由）を確認するだけで、商談の方向性を事前に立てられる。

## C-10. 新人営業マンへのメッセージ

**ポテンシャル承認**：10人面接に来て受かるのは1〜2人。採用されている時点でポテンシャルは認められている。
